Detailed Description

Root-finding algorithm implementing the Newton–Raphson method.

This class provides an implementation of the Newton–Raphson method to find roots of a nonlinear system defined by a Function. It iteratively updates the guess using the derivative (Jacobian) until convergence or a maximum number of iterations.
