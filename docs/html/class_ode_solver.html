Detailed Description

Abstract base class for iterative Ordinary Differential Equation solvers.

The OdeSolver manages the simulation loop, time tracking, and state updates. It holds a reference to the problem definition (Ode) and uses a specific numerical method (implemented in derived classes via Step()) to advance the solution.

virtual DynamicTensor OdeSolver::Step ( )
pure virtual

Calculates the solution at the next time step.

This is a pure virtual function that must be implemented by derived classes. It uses the specific numerical algorithm (e.g., RK4, Euler) to calculate y(t + dt) based on y(t).
