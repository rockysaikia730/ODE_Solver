Detailed Description

A class to represent a mathematical tensor of arbitrary rank at runtime.

The DynamicTensor stores data in a contiguous 1D memory block (std::vector) but provides an interface to access elements using N-dimensional coordinates. It uses Row-Major ordering (last dimension changes fastest).
