Detailed Description

Concrete implementation of Function using muParserX for parsing mathematical expressions. The documentation of muParserX can be found at: https://beltoforion.de/en/muparserx/ This class allows users to define ODE functions using string expressions that are parsed and evaluated at runtime. It supports multiple output expressions and can handle tensor inputs.
