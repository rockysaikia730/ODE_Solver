\doxysubsection{Detailed Description}
Explicit Runge–\+Kutta ODE solver. 

This class implements a classical 4th-\/order Runge–\+Kutta method for solving ordinary differential equations. It is an explicit solver suitable for non-\/stiff problems, providing higher accuracy than Forward Euler.
