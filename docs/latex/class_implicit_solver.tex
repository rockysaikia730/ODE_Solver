\doxysubsection{Detailed Description}
Multistep ODE solver that performs implicit time-\/stepping using a root-\/finding algorithm. 

This class implements an implicit multistep solver for ordinary differential equations (ODEs). At each time step, a nonlinear system arising from the implicit scheme is solved using a provided \doxylink{class_root_finder}{Root\+Finder} instance.
