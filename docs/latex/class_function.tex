Abstract base class representing a mathematical function for ODEs. This class defines the interface for evaluating functions that take a time and a \doxylink{class_dynamic_tensor}{Dynamic\+Tensor} as input and return a \doxylink{class_dynamic_tensor}{Dynamic\+Tensor} as output. Anyone overriding this class must implement the Eval method, and we also suggest to perform a check on dimensions inside the overridden method. The gradient method should also be overridden if the true gradient is known. The current gradient method uses central difference to approximate the gradient, only for real-\/valued, scalar functions.
