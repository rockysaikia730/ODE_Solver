\doxysubsection{Detailed Description}
Explicit multistep ODE solver implementing the Adams–\+Bashforth method. 

This class provides an explicit multistep method for solving ordinary differential equations. The Adams–\+Bashforth scheme uses previous derivative evaluations to compute the next state without requiring implicit root-\/solving, making it efficient for non-\/stiff systems.
