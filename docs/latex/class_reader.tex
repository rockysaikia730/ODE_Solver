\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Reader$<$ T $>$}
Abstract base class for reading ODE input data from files. 

This class defines a common interface used by all reader types (e.\+g., CSV reader, TXT reader, JSON reader).

The \doxylink{class_reader}{Reader}\+:
\begin{DoxyItemize}
\item Stores a filename and a field separator character.
\item Provides functions to check file existence and file extension.
\item Returns parsed data as either a \doxylink{class_function}{Function} or a \doxylink{struct_table_entry}{Table\+Entry$<$\+T$>$}.
\end{DoxyItemize}
