\doxysubsection{Detailed Description}
A class responsible for reading and parsing data from various sources. 

The \doxylink{class_reader}{Reader} class provides methods to read data from files, databases, or other input streams, and convert them into \doxylink{class_dynamic_tensor}{Dynamic\+Tensor} objects for further processing.
