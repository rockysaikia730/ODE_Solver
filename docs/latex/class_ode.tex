\doxysubsection{Detailed Description}
Represents an Ordinary Differential Equation (ODE) problem definition. 

The \doxylink{class_ode}{Ode} class acts as a container for the problem definition. It holds\+:
\begin{DoxyEnumerate}
\item The Initial Value Problem (IVP) conditions (t0, y0).
\item The physics/math function defining the derivative (dy/dt).
\item A root finding strategy (used for implicit solving methods). 
\end{DoxyEnumerate}
