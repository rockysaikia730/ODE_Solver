\doxysubsection{Detailed Description}
Abstract base class for iterative Ordinary Differential Equation solvers. 

The \doxylink{class_ode_solver}{Ode\+Solver} manages the simulation loop, time tracking, and state updates. It holds a reference to the problem definition (\doxylink{class_ode}{Ode}) and uses a specific numerical method (implemented in derived classes via \doxylink{class_ode_solver_a7a8a1ae3eb6213f9656a9b47ebbbe4fe}{Step()}) to advance the solution.
