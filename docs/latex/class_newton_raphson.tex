\doxysubsection{Detailed Description}
Root-\/finding algorithm implementing the Newton–\+Raphson method. 

This class provides an implementation of the Newton–\+Raphson method to find roots of a nonlinear system defined by a \doxylink{class_function}{Function}. It iteratively updates the guess using the derivative (Jacobian) until convergence or a maximum number of iterations.

Solves for the root of a nonlinear function using Newton–\+Raphson iterations. 

The method updates the initial guess according to\+:   \[    y_{k+1} = y_k - J^{-1}(y_k) F(y_k)
\] where $J$ is the Jacobian of $F$ at $y_k$.
