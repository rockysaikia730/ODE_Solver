\doxysubsection{Detailed Description}
Simple explicit first-\/order ODE solver using the Forward Euler method. 

This class implements the Forward Euler method, a basic explicit solver for ordinary differential equations. It is suitable for non-\/stiff problems and advances the solution by a single fixed step at a time.
