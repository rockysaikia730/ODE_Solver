\doxysubsection{Detailed Description}
Nonlinear equation representing the implicit update rule at a single time step. 

The implicit multistep method produces an equation of the form\+:   \[    F(t_{n+1},y_{n+1}) = 0
\] This class provides the evaluation and finite-\/difference gradient needed by the \doxylink{class_root_finder}{Root\+Finder}.
