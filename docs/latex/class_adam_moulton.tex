\doxysubsection{Detailed Description}
Implicit multistep ODE solver implementing the Adams–\+Moulton method. 

The Adams–\+Moulton family consists of implicit linear multistep methods. Unlike Adams–\+Bashforth, these methods require solving a nonlinear system at each step but generally offer higher stability and accuracy for stiff or moderately stiff ODEs. This class uses the \doxylink{class_implicit_solver}{Implicit\+Solver} interface along with a \doxylink{class_root_finder}{Root\+Finder} to perform each implicit solve.
