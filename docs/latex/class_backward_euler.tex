\doxysubsection{Detailed Description}
Implicit first-\/order ODE solver using the Backward Euler method. 

The Backward Euler method is a special case of the Backward Differentiation Formula (BDF) with order 1. It is an implicit method suitable for stiff ODEs, requiring a \doxylink{class_root_finder}{Root\+Finder} to solve the nonlinear equation at each time step.
