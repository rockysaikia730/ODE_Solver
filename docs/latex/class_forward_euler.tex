\doxysubsection{Detailed Description}
Explicit first-\/order ODE solver using the Forward Euler method. 

The Forward Euler method is a special case of the Adams–\+Bashforth method with order 1. It is suitable for non-\/stiff ODEs and advances the solution explicitly by a single step\+:   \[    y_{n+1} = y_n + h f(t_n, y_n)
\]
